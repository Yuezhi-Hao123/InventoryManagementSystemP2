# Inventory Management System Project 2

This project is a Java Swing inventory management system for COMP603 / ENSE600 Project 2.

## Main Features

- Add product
- View all products
- Search product
- Update product
- Delete product
- Update stock quantity
- View low-stock products
- Save product data using Apache Derby embedded database

## How to Run

1. Open the project in NetBeans.
2. Run `main--Main.java` to start the GUI.
3. Use the buttons in the GUI to manage products.
4. Run `ProductDAOTest` to run the JUnit 4 tests.

## Database

The system uses Apache Derby embedded database and JDBC.
